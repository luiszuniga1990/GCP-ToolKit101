# About 101 ToolKit GCP

## Vision
The **101 ToolKit GCP** is designed to be a comprehensive, yet accessible, resource for engineers and architects looking to implement best-in-class solutions on Google Cloud Platform. 

Whether you are just starting with Cloud Infrastructure or you are a seasoned professional looking for clean, modular templates, this toolkit provides "ready-to-deploy" cases that bridge the gap between basic automation and complex production environments.

## Core Pillars

### 1. Infrastructure as Code (IaC)
Leveraging **Terraform**, we provide modular blueprints for:
- **Data Engineering**: BigQuery datasets and automated ingestion pipelines (GCS to BQ).
- **Observability**: Centralized Log Sinks and custom Monitoring Dashboards.
- **Networking**: Secure VPC architectures with Private Google Access, Cloud NAT, and IAP.

### 2. Security Operations (SecOps)
Integrating security from the ground up with:
- **YARA-L Rules**: For advanced threat detection in Google Security Operations (formerly Chronicle).
- **SCC Configurations**: Best practices for Security Command Center.

## Why use this toolkit?
- **Modular**: Each case is independent. Use only what you need.
- **Production-Ready**: Follows Google Cloud Architecture Framework and security best practices.
- **Educational**: Clear documentation to help you understand the *why* behind every resource.
